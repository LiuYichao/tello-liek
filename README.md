## 安装

+ [node](https://nodejs.org/en/)
+ [yarn](https://classic.yarnpkg.com/en/docs/install)

下载完上述两个软件后，进入项目目录，输入
```
yarn install
yarn start
```

即可运行

## 功能列表
+ 注册，登录
+ 多任务列表添加，任务列表名称可编辑，删除
+ 任务卡片添加，编辑，删除，详情浏览
+ 任务成员增加，删除
+ 任务设施期限日
